Technical Challenges

01. Latency

Achieving real-time performance on constrained hardware required rewriting the core A* algorithm in Rust and utilizing manual memory management to eliminate garbage collection pauses during critical flight maneuvers.

02. Sensor Fusion

Integrating noisy LIDAR data with optical flow sensors created conflicting positioning signals. We implemented an Extended Kalman Filter (EKF) to smooth trajectory estimation and improve location accuracy by 40%.

03. Scalability

Designing the system to handle swarms meant the communication protocol had to be stateless. We moved from a centralized command server to a peer-to-peer mesh network architecture for resilience.
